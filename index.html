Object Detection

Object detection is a classic Computer Vision problem and one of the more efficient ways to solve it is with Single Shot Detectors (SSD). The goal of this project was to build an SSD using a Fully Convolutional Network (FCN). As a kid from the 90s I found an interesting dataset of about 1000 images of Pikachu to train on. The result is a working SSD that can successfully detect multiple objects on a test image with training time taking less than 4 minutes on GPUs.
